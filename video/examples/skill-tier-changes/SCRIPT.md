# 四战法改版，谁真正起飞？

规格：9:16 竖屏；普通话旁白。

## 口播文本

好的，我们来讲一下7月15日战法更新对演武的影响。

先说以静制动。
改版前，这个战法的评级是 T3。大数据记录了203场，胜率46.8%，表现其实不算差。
改版后，减伤的基础数值从30%降到20%，但保护目标从自身一人，扩大到自身加一名随机队友，而且减伤受到统率影响。假设实战中可以提升到28%，那么新版两名目标合计提供56%的减伤预期；相比旧版一人的30%，团队收益接近翻倍。
兵刃、追击、谋略和主动减伤又属于独立乘区，叠加后的实际收益还会更高。具体算法可以看我上一期关于减伤公式的视频。
再和T1+的步步为营比较。步步为营基础提供24%全类型减伤，按照相同的统率加成估算，可以提高到大约33.6%。它八回合的平均发动概率是87.5%，换算后的减伤预期大约是29.4%。新版以静制动则是28%稳定生效，两者已经非常接近。
步步为营的优势是全类型减伤，以静制动则可以利用多个独立减伤乘区。特别是偶数回合降低谋略和主动战法伤害，而演武里到处都是谋略和主动战法，所以实际保护能力并不差。
从强度来看，新版以静制动已经可以作为步步为营的低一档平替。
所以最终评级从原来的 T3上调到 T1。

下面说风卷残云。
改版前，这个战法的评级是 T4。大数据目前只有4场，胜率75%，样本太少，参考价值比较有限。
旧版对敌军随机两人分别造成180%兵刃伤害。目标处于风暴状态时，只提升这一次伤害，打完就结束。
改版后，单体伤害从180%提高到190%，总伤害从360%提高到380%。更重要的是，风暴目标不再只提升本次伤害，而是让自身造成伤害提升15%，持续两回合，并且可以叠加两层。
叠满以后就是30%增伤。风卷残云也因此从一个单次爆发战法，变成了增伤战法。
再和T1+的及锋而试比较。两者的发动率都是60%。及锋而试攻击两名目标，单体伤害是160%，并提供20%兵刃增伤；风卷残云的单体伤害是190%，在风暴体系中还能叠出30%全类型增伤。如果自身也处于风暴状态，还能额外获得10%规避率。
在能够稳定施加风暴的阵容里，新版风卷残云已经可以作为及锋而试的体系平替；但离开风暴体系，增伤就不够稳定。
所以最终评级从原来的 T4上调到 T2；在完整的风暴阵容中，可以按照 T1使用。

下面说舍生取义。
改版前，这个战法的评级是 T3。大数据一共记录了132场，胜率31.1%，整体表现比较一般。
舍生取义可以在前两回合为我军全体提供30%连击率，并让武力最高的队友获得清醒。旧版的问题是，携带者自己会受到震慑，前两回合完全无法行动。
改版后，自身受到的控制从震慑变成缴械。虽然还是不能普攻，但已经可以正常发动主动战法。
这个改动最大的意义，就是舍生取义终于可以和无难之志一起携带了。舍生取义提供全队30%连击率，无难之志再为两名队友提供45%连击率和20%倒戈。两名追击核心的连击率可以直接叠到75%。
对于演武这种阵容和战法都需要拼拼凑凑的环境，这套组合非常实用。即使没有完整的追击体系，也可以只用两个战法快速补足连击率、清醒和倒戈。
但问题是，目前追击战法在演武里的整体表现比较低迷。舍生取义虽然解决了自身罚站的问题，却没有改变追击体系的整体弱势。
所以最终评级从原来的 T3上调到 T2，但暂时没有提升到 T1。

下面说乘虚而入。
改版前，这个战法的评级是 T4。大数据一共记录了55场，胜率25.5%，表现比较差。
乘虚而入的伤害没有变化，仍然是对随机单体造成280%兵刃伤害。目标已经处于虚弱状态时，本次伤害提升30%。
这次改版唯一的加强，是对未虚弱目标施加虚弱的概率从65%提高到80%。
但这个战法本身只有40%发动率。换算下来，每次普通攻击成功施加虚弱的概率从26%提高到32%，实际只增加了6个百分点。
和同为T4的攻其不备相比，两者都是40%发动率和280%伤害。攻其不备可以稳定施加技穷，目标已有技穷时还有机会施加震慑；乘虚而入只有80%概率施加虚弱，控制能力并没有明显胜出。
所以新版乘虚而入勉强可以当成一个追击控制战法，但整体强度没有发生质变。
最终评级保持 T4不变。
